O arquivo header contem a declaração de todas as classes e métodos, e para que sejam utilizados em um sistema basta a inclusão desse arquivo. Caso seja necessario testar o funcionamento das classes domínio, esse arquivo deve ser incluido. Os métodos checam se os dados recebidos correspondem com o esperado, e em caso de tipo invalido, é lançada excessão avisando o usuário do erro.
